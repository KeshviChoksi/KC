# AirBNB SF Listings

Displays first 50 Airbnb listings from San Francisco using JavaScript fetch and async/await.

## Live Demo
https://keshvichoksi.github.io/KC

## Features
- Loads 50 listings dynamically from JSON
- Shows name, description, amenities, host, price, thumbnail
- Superhost badge as creative addition
- Built with Bootstrap 5 + Vanilla JS
